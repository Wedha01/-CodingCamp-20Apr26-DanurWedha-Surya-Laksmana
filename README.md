🚀 Daily Flow Dashboard
A minimalist, high-performance productivity dashboard designed to help you stay focused and organized. This is a standalone web application that runs entirely in your browser, storing all your data locally for maximum privacy and speed—no servers, no accounts, no fuss.

✨ Features
Adaptive Greeting: Dynamic greeting that changes based on the time of day.

Focus Timer: A built-in 25-minute Pomodoro timer with start, stop, and reset functions.

Smart To-Do List:

Auto-Sort: Tasks are automatically organized—active tasks stay at the top while completed ones move to the bottom.

Duplicate Prevention: A visual "shake" animation and warning message appear if you try to add a task that already exists.

Inline Editing: Edit your tasks directly by clicking on the text.

Quick Links: A customizable bookmarking system for your most-visited websites.

Theme Toggle: Switch between Dark and Light modes (your preference is saved automatically).

Privacy First: All data is stored via the Browser localStorage API. Your data never leaves your computer.

🛠️ Tech Stack
HTML5: Semantic structure.

Tailwind CSS: Modern, responsive utility-first styling.

Vanilla JavaScript: Lightweight logic with zero external dependencies.

LocalStorage: Client-side data persistence.

🚀 How to Use
Since this project requires no backend and no installation, getting started is instant:

Download the index.html file.

Open the file in any modern web browser (Chrome, Firefox, Edge, or Safari).

Bookmark the page (Ctrl+D / Cmd+D) to use it as your daily productivity hub.

⚙️ Customization
You can easily modify the dashboard by editing the index.html file:

Timer Duration: Change the timerSeconds variable (default is 1500 for 25 minutes).

Default Links: Modify the links array in the script section to include your own favorite starter sites.
